# Definition of Done

Bir gelistirme maddesi tamamlanmis sayilmasi icin:

- [ ] Kabul kriterleri tamamlandi
- [ ] Kod review tamamlandi
- [ ] CI kontrolleri basarili
- [ ] Gerekli testler eklendi/guncellendi
- [ ] Dokumantasyon guncellendi
- [ ] Hata ve kenar durumlar test edildi
- [ ] Release notuna etkisi eklendi
- [ ] Guvenlik/izolasyon etkisi degerlendirildi
- [ ] Operasyon etkisi (monitoring/rollback) notlandi
- [ ] Quality gate komutlari lokalde dogrulandi
